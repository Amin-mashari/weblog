# وبلاگ خبری
این پروژه برای یادگیری جنگو نوشته شد.
این پروژه یک سایت مقالات است که شما می‌توانید در آن مقالات موجود از نویسنده‌های مختلف را مشاهده کنید، مانند سایت های خبری همچون ورزش۳.

در این پروژه کاربر های مختلفی همچون مدیرسایت، نویسنده، کاربرویژه، کاربرعادی و میهمان وجود دارند.

# منابع

کانال [یوتیوب سیلیسیم](https://gitlab.com/mtarif/django-course-source)، مدرس: مهران کامل

سایت اصلی [جنگو](https://docs.djangoproject.com/en/4.1/)
